# hasfun
IT程序员编程时解决困难平台

```
hasfun-pom                   - - - - 父模块
    |---hcw-common           - - - - 常用工具类
    |---hcw-net-framework    - - - - 基于netty的RPC网络框架
    |---hcw-netty-springmvc  - - - - 基于netty的springMVC
    |---hasfun-db            - - - - 数据层
    |---hasfun-page          - - - - 静态页面
    |---hasfun-web           - - - - hasfun网站http服务接口层
    |---hasfun-question      - - - - 问答RPC业务服务层
```
